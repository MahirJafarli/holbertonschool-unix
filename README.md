## File Descriptions

| File | Description |
| :--- | :--- |
| **0-hellofile** | Displays the entire content of the `/etc/passwd` file. |
| **1-lastlines** | Displays the last 10 lines of the `/etc/passwd` file. |
| **2-firstlines** | Displays the first 10 lines of the `/etc/passwd` file. |
| **3-third_line** | Displays only the 3rd line of the file `iacta` (using a pipe). |
| **4-cwd_state** | Writes the output of `ls -la` into a file named `ls_cwd_content`, overwriting any existing content. |
| **5-hidethisword** | Displays all lines in `/etc/passwd` that **do not** contain the word "bin". |
